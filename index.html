What is Tetris2?
Tetris2 is the Tetris you've always known, but co-op! Using our download link, play 2-player Tetris on your computer!
Some random information.

How is local co-op implemented?

To implement local co-op, we we have both players on the same keyboard. Player one uses the arrow keys, while player 2 uses the wasd keys.

How does the game keep track of scores?

Player1 will first login, and then begin playing. Only player1's score will be recorded. Player1's score will be increased when they clear a row using their key.

Where can I see the highscores list?

The highscore list will be kept on whatever directory the game was installed on. It will be in a txt file named 'score.txt'

How can I play?

To play, use the download link this page provides. After downloading, run main.py. Note that you must have python3 and pygame installed first.
